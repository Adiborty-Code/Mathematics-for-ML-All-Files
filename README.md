# Mathematics for Machine Learning and Data Science

This repository contains all the files, notes, and code used while completing the **"Mathematics for Machine Learning and Data Science"** course on Coursera.

## About the Course

This specialization covers the fundamental mathematics needed for machine learning and data science, including:

- **Linear Algebra** - Vectors, matrices, and their operations
- **Calculus** - Derivatives, gradients, and optimization
- **Probability & Statistics** - Distributions, inference, and statistical analysis

  
## Technologies Used

- Python
- NumPy
- Matplotlib
- Jupyter Notebooks

## Course Information

- **Platform**: Coursera
- **Specialization**: Mathematics for Machine Learning and Data Science
- **Offered by**: DeepLearning.AI
